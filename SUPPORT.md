# Support noalphjs

## Cara Mendapatkan Bantuan

- **Bug report:** [GitHub Issues](https://github.com/AlphaIsYour/noalphjs/issues)
- **Diskusi fitur:** [GitHub Discussions](https://github.com/AlphaIsYour/noalphjs/discussions)
- **Pertanyaan umum:** [GitHub Discussions → Q&A](https://github.com/AlphaIsYour/noalphjs/discussions/categories/q-a)

## Sebelum Membuka Issue

1. Cari di Issues yang sudah ada
2. Cek dokumentasi di `/docs`
3. Pastikan kamu menggunakan Node.js >= 20

## Template Issue

Gunakan template yang tersedia saat membuka issue baru.
